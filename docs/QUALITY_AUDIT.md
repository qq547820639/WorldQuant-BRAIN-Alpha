# Module Quality Audit

## Generator

生成器是模板库驱动，适合安全测试和运营闭环。生产质量取决于后续接入更强的假说生成器和数据字段库。

## Local Prefilter

低质量候选不会直接打官方接口。预筛输出会进入候选归档，并保留拒绝原因。

## Official API

官方 API 与 mock API 分离。生产环境必须显式选择，并且提交前调用官方 alpha check。

## Scoring

评分分为先验、官方经验指标、提交前检查卡三部分，并保留校准字段。

## Gate

生产门禁只回答“是否值得提交”。账户安全门禁单独回答“是否允许提交，尤其是否允许自动提交”。

## Repository

所有候选、事件、家族记录、A/B 测试记录都写 JSONL。凭据不入库。

## Known Limits

- 本地系统无法知道真实跨用户相关性，必须依赖官方返回。
- 阈值是经验参考，不是官方承诺。
- 一次运行不能保证生成高质量 Alpha，只能保证研究流程可审计、可校准、尽量保护账户。
